# 🥋 Taekwondo System - Frontend

## 📌 Descripción
Frontend de un sistema web para la gestión de escuelas de taekwondo.  
Permite la administración de alumnos, control de información y conexión con un backend mediante API.

---

## 🚀 Tecnologías utilizadas

- ⚡ Angular y React
- 🟦 TypeScript
- 🎨 HTML5 & CSS3
- 🔗 Consumo de API REST

---

## 🎯 Funcionalidades

- 👤 Gestión de alumnos
- 📝 Formularios para registro y edición de datos
- 🔍 Visualización de información en tiempo real
- 🔐 Integración con backend para manejo de datos

---

## 🧱 Estructura del proyecto

- `components/` → Componentes de la aplicación  
- `services/` → Servicios para conexión con API  
- `models/` → Estructuras de datos  
- `pages/` → Vistas principales
